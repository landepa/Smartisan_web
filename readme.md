## 项目说明

**html目录结构：**

+ html
  + header.html    	  ==>    公共头部
  + title_nav.html        ==>    公共导航栏
  + footer.html	        ==>	公共底部
  + **index.html**	       ==>	商城首页
  + goodList.html       ==>    商品列表页
  + goodDetail.html   ==>    商品详情页
  + login.html              ==>    登录页面

**页面跳转：**

点击页面右上角跳转到登录页面

点击顶部白色背景导航栏跳转到商品列表页

点击商品跳转到商品详情页

说明：登录页输入符合要求的账号和密码即可登录。
